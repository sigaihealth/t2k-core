# T2K Specification

The current normative version is [T2K Ontology Pack Specification 1.0](v1.0/README.md).

Each version has immutable prose, schema, and conformance expectations. The
canonical machine-readable schema is published at:

```text
https://t2k.ai/schemas/t2k-ontology-pack.v1.schema.json
```

Repository changes follow [GOVERNANCE.md](../GOVERNANCE.md). A future version
must not silently change the meaning of the v1 schema identifier.
